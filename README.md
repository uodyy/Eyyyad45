# 🎙️ Discord Voice Joiner & Hanger

A discord tool designed to help you join and stay in Discord voice channels using multiple accounts simultaneously. It supports essential features like **Mute**, **Deaf**, and **Streaming** to simulate real activity.

> **Visual Preview:** *Refer to the screenshots in the repository to see the interface and features in action.*

---

## 🚀 Installation & Usage Guide
Follow these steps carefully to ensure the tool operates correctly.

### Step 1: Install Required Libraries
Open your Terminal (CMD or PowerShell) in the project directory and run the following command to install all dependencies:

```bash
pip install -r requirements.txt
```
### Step 2: Configuration
You need to configure your settings before launching the tool:
1. **config.json**: Open this file and insert your **target Voice Channel ID**.
2. **tokens.txt**: Add your Discord account tokens here **(one token per line)**.

### Step 3: Launch the Tool
Run the main script using the following command:
```bash
python main.py
```
